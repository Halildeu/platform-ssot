# ADR-001: OpenFGA for Authorization (Zanzibar Model)
**Status:** Accepted  **Date:** 2026-03-29

## Decision
Use OpenFGA (Apache 2.0, Zanzibar) for authorization. Keycloak = authn only. Permission-service REMOVED.

## Consequences
Positive: Zanzibar inheritance, OSS, Java/JS SDK. Negative: New tech learning curve.
