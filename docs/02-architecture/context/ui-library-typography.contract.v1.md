# UI Library Typography Contract v1

Kanonik kaynak:
`docs/02-architecture/context/ui-library-typography.contract.v1.json`

## Amaç

UI kit icindeki metin dilini semantic olarak kilitlemek:

- display
- heading
- title
- body
- meta

## Kurallar

- Varsayilan semantic owner `Text` componentidir.
- Dar alanda uzun metin clamp/truncate/wrap karari olmadan birakilmaz.
- Sayisal verilerde `tabularNums` owner API ile kullanilir.
- Heading seviyesi gorunum degil bilgi hiyerarsisi icin secilir.
