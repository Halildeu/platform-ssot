# 03-delivery

Bu klasör geliştirme sürecinin yürütme alanıdır.

Alt içerikler:
- STORIES/        → STORY-*.md
- ACCEPTANCE/     → Given–When–Then kabul kriterleri
- TEST-PLANS/     → test stratejileri
- PROJECT-FLOW.md → story durumu ve roadmap

Amaç: “İş nasıl ilerliyor?” sorusuna cevap vermektir.

Notlar:
- Story/Acceptance/Test Plan ID’leri NUMARALANDIRMA-STANDARDI.md dokümanındaki
  formatlara göre verilmelidir (STORY-XXXX, AC-XXXX, TP-XXXX).  
- Genel doküman yerleşimi için docs/00-handbook/DOCS-PROJECT-LAYOUT.md
  her zaman ana referans kabul edilir.
