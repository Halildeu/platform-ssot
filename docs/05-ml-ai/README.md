# 05-ml-ai

Bu klasör AI/ML modelleri ve veri setleri için dokümantasyon alanıdır.

Alt içerikler:
- DATA-CARDS/         → veri seti açıklamaları (DATA-*.md)
- MODEL-CARDS/        → model kartları (MODEL-*.md)
- EVALUATION-REPORTS/ → model değerlendirme raporları (EVAL-*.md)

Amaç: AI modellerinin izlenebilirliğini ve hesap verebilirliğini sağlamak.
