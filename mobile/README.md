# Platform Mobile

Expo application scaffold for the managed repo mobile workspace.

## Start

- `npm run start`
- `npm run android`
- `npm run ios`
- `npm run web`

## Checks

- `npm run typecheck`
- `npm run doctor`

## Layout

- `app/` application root and providers
- `screens/` route-level composition
- `features/` feature-scoped UI and hooks
- `services/` API, storage, network, device, notifications
- `store/` app state bootstrap
- `theme/` shared design tokens
- `packages/mobile-tokens` shared design tokens package
- `packages/mobile-ui` reusable presentational component library
- `packages/mobile-core` shared hooks, types, and service adapters
