## API Sözleşmeleri

Amaç
- OpenAPI/contract ve entegrasyon anlaşmalarını merkezi olarak tutmak.

Stil Rehberi
- Tüm API dokümanları için resmi stil standardı: `docs/00-handbook/STYLE-API-001.md`

Mevcut API Dokümanları
- `users.api.md` — kullanıcı listeleme, gelişmiş filtre, sıralama, sayfalama
- `auth.api.md` — login ve token claim sözleşmesi
- `audit-events.api.md` — audit event listeleme, export ve SSE
- `common-headers.md` — ortak header ve güvenlik scope’ları

Diğer
- `openapi/` klasörü — OpenAPI tanımları
- `postman/`, `insomnia/` — örnek istemci koleksiyonları
