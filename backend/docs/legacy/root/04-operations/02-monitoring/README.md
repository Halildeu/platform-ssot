# Observability Playbook

Bu klasör, platformdaki mikro servis ve mikro-frontend’lerin izlenebilirlik ayak izlerini belgelemek içindir. Her modül için kullanılan metrikler, log sorguları, panolar ve alarm koşulları burada tutulur. Yeni dashboard oluştururken bu rehberdeki formatı izleyin.

- `security-dashboard.md` — Security modülü ve perimeter metrikleri
- (arşiv) `archive/2025-11-04-security-dashboard-improvements.md` — Security panel geliştirme planı (tamamlandı)
- `assets/README.md` — Snapshot ve paylaşım arşivi
- (planlı) `audit-dashboard.md` — Audit modülü ve event stream takibi
