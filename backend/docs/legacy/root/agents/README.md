# Agents – Security & Automation Specs

Bu klasör, güvenlik/compliance ve otomasyon için kullanılan:
- Normatif kural dokümanlarını (`docs/agents/*.md`),
- Ve bu kuralları uygulayan **runtime agent / araç SPEC’lerini** (`SPEC-AGENT-*.md` formatı)
barındırır.

Yönetici agent:
- Güvenlikle ilgili tüm işler için referans agent: `docs/agents/AGENT-SECURITY.md`
- Bu dosya, hangi kural dokümanlarının MUST olduğunu ve LLM agent’ın nasıl davranması gerektiğini tanımlar.

Format MP’si:
- Agent formatları ve örnek iskeletler için standart kaynak:  
  - `docs/agents/MP-AGENT-FORMAT.md`

Bu README, klasörün amacını ve içeriğini özetler;  
**agent dokümanlarının nasıl yazılacağı** için her zaman `MP-AGENT-FORMAT.md` referans alınmalıdır.
