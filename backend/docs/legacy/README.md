# backend/docs/legacy

Bu klasör, yeni `docs/` hiyerarşisine taşınmış dokümanların eski kopyalarını
arşivlemek için kullanılır.

- Aktif ve güncel tüm dokümanlar `docs/` altında yaşar.  
- Buradaki içerikler yalnızca tarihçe / referans amaçlıdır.  
- Yeni doküman üretimi için:  
  - NUMARALANDIRMA-STANDARDI.md  
  - docs/00-handbook/DOCS-PROJECT-LAYOUT.md  
  - docs/03-delivery/PROJECT-FLOW.md  
rehber alınmalıdır.
