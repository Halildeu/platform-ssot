## Backend Docs (Legacy Kök)

Aktif ve güncel dokümanlar artık `docs/` hiyerarşisi altında tutulmaktadır.
Bu `backend/docs` ağacı **yalnızca legacy / arşiv** amaçlıdır.

Yeni dokümantasyon için:
- NUMARALANDIRMA-STANDARDI.md  
- docs/00-handbook/DOCS-PROJECT-LAYOUT.md  
- docs/03-delivery/PROJECT-FLOW.md  
rehber alınmalıdır.

Taşıma Süreci:
- STORY-0001-backend-docs-refactor  
- AC-0001-backend-docs-refactor  
- ADR-0001-backend-docs-reorg  
ve `docs/02-architecture/services/backend-docs/TECH-DESIGN-backend-docs-migration.md`
dosyalarında tanımlanan plana göre ilerlemelidir.

Not:
- Taşıma tamamlandıkça eski dosyalar `backend/docs/legacy/` altına
  alınmalı ve burada arşivlenmelidir.
